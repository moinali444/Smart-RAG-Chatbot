# 🤖 Smart RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDFs and ask questions.

## 🚀 Features
- PDF upload
- Semantic search using FAISS
- AI-generated answers using local LLM (Gemma via Ollama)

## 🛠 Tech Stack
- Python
- Streamlit
- FAISS
- Sentence Transformers
- Ollama (Gemma)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python -m streamlit run rag_app.py
